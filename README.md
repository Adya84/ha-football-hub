# Football Hub

Football Hub is a Home Assistant custom integration for live football fixtures, results, league tables, match stats, line-ups and competition data.

This integration is designed to support multiple football competitions from one codebase, starting with league-based competitions such as the Premier League.

## Features

- Live match centre
- Fixtures and results
- League tables
- Match statistics
- Team centre
- Top scorers and player stats
- Line-ups and match events
- Home Assistant dashboard panel
- HACS support
- Multi-competition structure planned

## Status

This project is in early development.

The first target competition is the Premier League, with support for more leagues and competitions planned.
